## **Pré-requisitos**
Certifique-se de ter os seguintes itens instalados em sua máquina:
1. **Python 3.8** ou superior
2. **Biblioteca Pygame**
   - Instale o Pygame com o seguinte comando:
     ```bash
     pip install pygame
     ```
